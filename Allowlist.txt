# CaptainQwerty's AdGuard Home Allowlist
# ===============================================
#
# Description: Curated allowlist of domains that should be allowed through AdGuard Home DNS filtering
# Purpose: Prevents blocking of essential services, streaming platforms, and legitimate websites
# Last Updated: April 18, 2026
# Repository: https://github.com/captainqwerty/CaptainQwertysAdGuardLists
#
# Usage:
# - Import this file into your AdGuard Home instance under Settings > DNS Settings > DNS allowlist
# - Domains marked with ^$important have highest priority and cannot be overridden by blocklists
# - Regular domains (without ^$important) can still be blocked by other rules if needed
#
# Organisation:
# - Domains are grouped by category for easier maintenance
# - Each section includes related services and their subdomains
# - Comments explain the purpose of each domain group
#
# Maintenance:
# - Remove duplicates and test changes before committing
# - Add new domains with appropriate categorisation
# - Update Last Updated date when making changes
